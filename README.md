## 📖 حول الأداة (About The Tool)
أداة احترافية ومطورة خصيصاً لبيئة **Termux**، تتيح للمستخدمين تحميل الملفات الفردية أو "قوائم الروابط المجمعة" دفعة واحدة وبسرعة فائقة. تقوم الأداة تلقائياً بفرز وتصنيف الملفات حسب امتدادها (صور، مستندات، ملفات مضغوطة، فيديوهات) ونقلها مباشرة إلى مجلد التحميلات الرئيسي للهاتف (`/sdcard/Download`) مع واجهة رسومية ملونة ومؤشر تحميل ذكي.

An advanced CLI automation tool designed for **Termux** to download single or bulk URLs with high efficiency. It automatically filters, categorizes, and routes downloaded files into systematic folders (Images, Documents, Compressed, etc.) directly inside your local phone storage.

---

## ✨ الميزات الأساسية (Key Features)
* 🚀 **تحميل مجمع (Bulk Downloader):** تحميل عشرات الروابط تلقائياً عبر ملف نصي واحد.
* 📂 **تنظيم تلقائي ذكي:** فرز وتصنيف الملفات بناءً على نوع الامتداد فوراً.
* 📊 **مؤشر تحميل ديناميكي:** شريط حركي يعرض النسبة المئوية للتحميل بدقة ثانية بثانية.
* 🎨 **واجهة سينمائية (Cyber UI):** تصميم صندوقي رقمي فخم بألوان متناسقة ومريحة للعين.
* 🗺️ **دعم كامل للغة العربية:** إصلاح مشكلة الخطوط المعكوسة والمتقطعة في ترمكس.

## 🛠️ طريقة التثبيت والتشغيل (Installation & Usage)

افتح تطبيق ترمكس وقم بتنفيذ الأوامر التالية بالترتيب:


# 1. تحديث الحزم الأساسية
pkg update && pkg upgrade -y

# 2. تثبيت بايثون وجيت
pkg install python git -y

# 3. جلب مستودع الأداة
git clone [https://github.com/dev-ABOODI-cpu/CyberOrganizer-Pro.git](https://github.com/dev-ABOODI-cpu/CyberOrganizer-Pro.git)

# 4. الدخول لمجلد الأداة
cd CyberOrganizer-Pro

# 5. تثبيت المكتبات المطلوبة بـأمر واحد
pip install -r requirements.txt

# 6. إعطاء صلاحية الوصول لذاكرة الهاتف (مهم جداً)
termux-setup-storage

# 7. تشغيل الأداة الفخمة
python organizer.py
